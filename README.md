# odin-recipes

A basic recipe website built with HTML as part of The Odin Project foundations curriculum.

The site consists of a main index page linking to three of my favorite recipes: sushi, truffle pasta, and nachos.

## Skills Demonstrated

- Structuring a multi-page website with HTML
- Using anchor tags to link between pages
- Organizing content with semantic HTML elements (headings, lists, images)
- Working with relative file paths
